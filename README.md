# DatatronInterface
建筑数字魔方项目接口

# 安装依赖

`pip install -r requirments.txt`

# 运行脚本

1. 命令行进入目录 `atd_test`
2. 运行unittest命令:
    
    1. 运行单个用例:

        `python -m unittest TestModule.TestClass.TestFunc`
    2. 运行类:

        `python -m unittest TestModule.TestClass`
    3. 测试发现:
    
        `python -m unittest discover -v`
